# Glosario oficial

**Estado:** Active  
**Versión:** 1.0.0

- **Financial Snapshot:** representación financiera determinista, versionada e inmutable para un alcance temporal.
- **Knowledge Collection:** conjunto canónico de hechos derivados de snapshots autorizados.
- **Insight:** observación explicable producida por una regla identificada y versionada.
- **AI Foundation:** frontera de privacidad, contexto y proveedores sobre la que se construyen capacidades de IA.
- **AI Interaction:** unidad neutral de trabajo inteligente solicitada por Private Balance, independiente de chat o proveedor.
- **Intent:** objetivo funcional declarado de una interacción.
- **Capability:** capacidad requerida para resolver una interacción sin nombrar un proveedor.
- **Interaction Policy:** restricciones aplicables a propósito, contexto, procesamiento y resultado.
- **Conversation:** secuencia ordenada de mensajes que utiliza una o más interacciones.
- **Session:** límite temporal y operativo que agrupa interacciones relacionadas.
- **Memory:** información explícitamente retenida para reutilización futura, con alcance y revocación.
- **Authorized Context:** contexto materializado únicamente después de superar la frontera de privacidad.
- **Privacy Boundary:** componente default-deny que decide si una solicitud puede procesarse.
- **Provider:** adaptador técnico capaz de ejecutar una solicitud compatible.
- **Contract:** API pública tipada que define comportamiento permitido sin imponer implementación.
- **Fail closed:** ante ausencia, ambigüedad o incompatibilidad, la operación se rechaza de forma segura.
